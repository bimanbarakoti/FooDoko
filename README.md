# FooDoko

A modern food delivery app built with Flutter.

## Features

- AI-powered food recommendations
- Voice ordering with speech recognition
- AR food visualization
- Real-time order tracking
- Biometric authentication
- Social features and group ordering

## Tech Stack

- **Frontend**: Flutter 3.24+
- **State Management**: Riverpod
- **Navigation**: GoRouter
- **UI**: Material Design with custom animations
- **Voice**: Speech-to-text integration
- **Authentication**: Biometric and social login

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Build

```bash
flutter build apk --release
```
